README — Vector Field Viewer
About this Visualisation

This viewer shows a 3D velocity field of the Taurus star-forming region, reconstructed from stellar proper motions and radial velocities using NIFTy (Numerical Information Field Theory). A regular grid is placed around the data, divided into voxels of 3 × 3 × 3 pc each — every voxel carries an inferred 3D velocity vector. A density mask derived from kernel density estimation of the stellar positions marks which voxels actually contain stars; this lets you isolate the physically relevant regions from the surrounding empty grid. The streamlines are purely a visualisation aid — they trace paths through the inferred velocity field and are not physical trajectories.

Getting Started

Run export_vectorfield.py to embed your data into this HTML file, then open vectorfield_viewer_data.html in any browser. No server needed.

Display Toggles
ArrowsShow/hide velocity arrows at each voxel center. Size and subsampling adjustable below.
StreamlinesShow/hide streamlines integrated through the field.
VoxelsShow/hide non-mask voxels (outside the stellar density mask).
MaskShow/hide mask voxels (inside the stellar density region).
GridShow/hide the bounding box wireframe.
AxesShow/hide labeled X/Y/Z axes with tick marks.
StarsShow/hide real stellar positions as spheres. Opens the Star coloring panel.
Mask onlyWhen on, only voxels inside the stellar density mask are drawn. Colorbar rescales to mask range.
Subtract v_refToggle bulk velocity subtraction. Off = raw field; On = field in the Taurus rest frame.
Streamline Modes
Random seedsRK4 traces seeded randomly inside the mask. Opacity encodes local speed.
Arrow networkEach mask voxel traces a short segment to its downstream neighbor — shows flow topology.
AnimatedParticles flow along pre-traced paths. Speed proportional to local field magnitude.
Streamline Sliders
CountNumber of streamlines / particles.
Max. lengthMaximum number of RK4 integration steps. Each step covers ~0.3 pc (fixed step size of 0.1 × voxel spacing), so length ≈ max steps × 0.3 pc. Increase for longer lines across the region.
Mask marginHow many voxels outside the mask a streamline may still travel before stopping. 0 = strictly inside mask.
OpacityGlobal opacity multiplier. Unlike other sliders, changing this never triggers a recalculation — it updates instantly via a GPU shader uniform.
Anim. speedPlayback speed for animated mode. Particles move faster in high-magnitude regions regardless of this setting.
Colormaps

Three independent colormaps can be set in the Appearance section — one each for the vector field, streamlines, and stars. All use the same set of options:

ViridisDark purple → blue → teal → green → yellow. Perceptually uniform, good default for magnitudes.
PlasmaDark blue → purple → pink → orange → yellow. Also perceptually uniform, slightly higher contrast than Viridis.
MagmaBlack → dark purple → red → cream. Good for highlighting sparse high-value regions against a dark background.
InfernoBlack → purple → orange → yellow. Similar to Magma but warmer; strong contrast at the high end.
CividisDark blue → teal → olive → yellow. Optimised for colour-blindness (no red/green distinction needed).
RdBuRed → white → blue. Diverging scale — ideal when the data has a meaningful zero (e.g. velocity relative to the bulk frame). White = zero, red = positive, blue = negative.
CoolwarmBlue → white → red. Similar to RdBu but smoother transitions; less saturated at the extremes.
TurboFull rainbow from dark blue to dark red via cyan, green, yellow. High overall contrast; good for seeing fine structure, but not perceptually uniform.

The velocity colorbar always rescales to the actual min/max within the mask when Mask only is active, so the full color range is used regardless of the global field extremes.

Arrow Sliders
SizeArrow length scale relative to voxel spacing.
SubsampleOnly draw every Nth voxel (1 = all, 2 = every 2nd, …). Reduces clutter and improves performance.
Voxel Sliders
OpacityTransparency of voxel cubes.
Mag. thresholdHide voxels with magnitude below this fraction of the maximum.
Clip Plane

Select an axis (X/Y/Z) and use the Position slider to show only a single slice through the volume. Useful for inspecting the interior. Set to Off to show everything.

Star Coloring

Appears when Stars is active. Choose a field from the dropdown to color stars by that property (age, rv, pmra, …). Stars without a value for the chosen field are hidden. A colorbar (Plasma colormap) shows the min/max range.

Performance Tips
  • Increase Subsample to reduce arrow count.
  • Reduce Count for fewer streamlines.
  • Turn off Arrows — each ArrowHelper is ~5 draw calls.
  • Turn off Voxels / Mask — 25 000 boxes are expensive.
  • Arrow network and Animated modes are cheaper than Random seeds at high counts.
